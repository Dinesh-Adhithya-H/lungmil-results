1System Overview

Two-phase pipeline for multimodal lung transplant rejection classification. Phase 1 trains one ABMIL model per modality with optional bidirectional cross-attention knowledge distillation (HE & Clinical → teacher; CT & BAL → student). Phase 2 fine-tunes all encoders jointly under six different multimodal fusion strategies.

5Phase 2 — Fusion Variants
Phase 2 · 200 epochs · lr=5e-5 · modal_dropout=30% · 11 variants (6 base + 5 _cls)

All variants share the same frozen P1 encoders (GatedAttentionEncoder, loaded from best_model.pt, fine-tuned during P2) and frozen ProjectionHeads. A _cls suffix replaces the ABMIL gate with a learnable CLS token + single MHA head for pooling.

Middle Fusion  / middle_cls
MiddleFusionMIL
Each modality is first compressed to a single 256-d summary via its own ABMIL encoder. The small set of summary vectors forms a sequence that is processed by a 2-layer cross-modal transformer. Final representation = mean of transformer outputs (or CLS token). Computationally efficient — transformer operates on n_mods tokens, not patches.

Iterative XModal  / iterative_cls
IterativeXModalMIL
Most complex variant. Applies R=2 iterative blocks at the raw patch level before slot compression. After R blocks, IterativeSlotAttn → CrossModalTransformer → pool. Supports gradient checkpointing for memory savings on long HE sequences.

6Loss Functions
Phase 1  Per-Sample Losses
L_task
Hinge loss with class weights (all modalities)
y = 2·label − 1 ∈ {−1, +1} L = w · max(0, 1 − y · logit) w_pos = N/(2·N_pos) w_neg = N/(2·N_neg) clamp max=20
L_cross
Aux head on cross-attended student rep (students only)
Hinge(cross_aux_head(r_stud_enr), label) λ_cx = 0.05
L_kd
Attention transfer KD — Zagoruyko & Komodakis ICLR 2017
Select top-K=50 student patches by alpha_a2b (teacher's view) KL( softmax(α_self[top-K]/τ=0.5) ‖ softmax(α_a2b[top-K]/τ) ) λ_kd = 0.05 — scale-invariant, relative ranking only
L_crd
CRD cosine distance — He et al. ICLR 2020
1 − cosine_similarity(r_stud_enr, r_teacher) ∈ [0, 2] Pulls cross-attended student summary toward teacher rep λ_crd = 0.1
L_clr
Supervised contrastive — Khosla et al. NeurIPS 2020 (optional)
Positives: same stem + diff modality OR same label + diff stem Negatives: different label Class-weighted by w_pos/w_neg τ = 0.07 λ = 0.1 computed at gradient-accumulation boundary
Phase 2  Training Losses
L_task
Same hinge loss with class weights as Phase 1
200 epochs · lr=5e-5 · weight_decay=1e-3 grad_accum=4 · AMP (mixed precision)
L_recon
Optional reconstruction loss (middle / crossattn / crossmodal / iterative)
Random modality m sampled per training sample L = MSE( decoder_m(rep), summary_m.detach() ) Decoder: Linear(256→256) → GELU → Linear(256→256)
Model selection: Best val_BAcc checkpoint is saved inline during training (written at every val_BAcc improvement to best_model.pt). No expensive end-of-training GPU rescan. Fast fallback: history-based ckpt selection from stored val_bacc lists.
